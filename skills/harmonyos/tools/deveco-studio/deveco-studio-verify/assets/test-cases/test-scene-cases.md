## 一、设备管理测试

### 1.1 设备列表查询测试

#### 用例1：列出已连接设备
**场景描述**：测试列出当前已连接的设备列表

**前置条件**：
- hdc 工具可用
- 至少一个设备已连接

**操作步骤**：
1. 执行 hdc list targets 命令
2. 解析设备列表输出
3. 提取设备ID（如 127.0.0.1:5555）
4. 记录设备数量

**期望结果**：
- 正确列出所有已连接设备
- 设备ID格式正确（127.0.0.1:端口号）
- 设备数量 ≥ 1

**示例输出**：
```
127.0.0.1:5555
127.0.0.1:5557
```

---

#### 用例2：列出已安装模拟器
**场景描述**：测试列出已安装的模拟器实例及其详细信息

**前置条件**：
- emulator 工具可用
- 至少一个模拟器已配置

**操作步骤**：
1. 执行 emulator -list 命令
2. 执行 emulator -list -details 命令
3. 解析 JSON 格式输出
4. 提取模拟器名称、设备类型、镜像版本

**期望结果**：
- 正确列出所有模拟器实例
- JSON 格式包含完整信息：
  - name: 模拟器名称
  - deviceType: 设备类型（Phone/Foldable/WideFold）
  - osVersion: 系统版本（如 HarmonyOS 6.0.2(22)）
  - isRunning: 运行状态
  - instancePath: 实例路径

**示例输出**：
```json
{
  "name": "Pura 9022",
  "deviceType": "phone",
  "osVersion": "HarmonyOS 6.0.2(22)",
  "isRunning": "false"
}
```

### 2.2 批处理文件启动测试 ✅

#### 用例9：创建临时批处理文件启动模拟器
**场景描述**：测试使用 PowerShell 创建临时批处理文件启动模拟器（推荐方法）

**前置条件**：
- PowerShell 可用
- 模拟器名称已知

**操作步骤**：
1. 创建临时批处理文件：
   ```powershell
   $batPath = "$env:TEMP\start_emulator.bat"
   @"
   @echo off
   cd /d "C:\Program Files\Huawei\DevEco Studio\tools\emulator"
   Emulator.exe -start "Pura 9022"
   pause
   "@ | Out-File -FilePath $batPath -Encoding ASCII
   ```
2. 在新窗口启动批处理文件：
   ```powershell
   Start-Process -FilePath $batPath -WindowStyle Normal
   ```
3. 等待 30 秒
4. 执行 `hdc list targets` 检查连接

**期望结果**：
- 批处理文件成功创建
- 模拟器在新窗口启动
- 设备成功连接（127.0.0.1:5555）
