# 测试用例 1：分析appfreeze的faultlog，生成问题分析报告

## 场景描述
用户希望分析appfreeze的faultlog，生成问题分析报告。

## 用户输入
请使用hmos-appfreeze-analysis skill，分析故障日志：xx.log 和采样栈：xx

## 执行步骤
1. 读取用户指定的文件
2. 按照skill工作流步骤执行
3. AI返回故障日志分析报告
