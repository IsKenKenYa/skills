# 实况窗 UI 布局规范

## 布局原则

1. **信息优先级**：重要信息左上，时间/状态右上
2. **简洁清晰**：避免信息过载
3. **响应式**：适配不同模板尺寸
4. **品牌一致**：与应用主色调一致

## 尺寸规格

| 类型 | 尺寸 |
|------|------|
| 短卡片 | 200vp × 80vp |
| 中卡片 | 200vp × 120vp |
| 长卡片 | 280vp × 160vp |
| 沉浸式 | 360vp × 200vp |

## 边距规范

- 容器内边距：12vp
- 元素间距：8vp
- 文字与图标间距：4vp

## 颜色规范

| 用途 | 颜色值 |
|------|--------|
| 主色调 | #F7B500 |
| 背景色 | #FFFFFF / #1A1A1A |
| 文字主色 | #000000 / #FFFFFF |
| 文字副色 | #666666 / #999999 |
| 状态成功 | #07C160 |
| 状态警告 | #FFC107 |
| 状态错误 | #F5222D |

## 圆角规范

| 尺寸 | 圆角半径 |
|------|----------|
| 小卡片 | 12vp |
| 中卡片 | 16vp |
| 大卡片 | 20vp |
| 沉浸式 | 24vp |

## 字体规范

| 元素 | 字号 | 字重 | 最大行数 |
|------|------|------|----------|
| 标题 | 16fp | Medium | 1 |
| 正文 | 14fp | Regular | 2 |
| 辅助文字 | 12fp | Regular | 1 |
| 状态标签 | 10fp | Medium | 1 |

## 图标规范

- 小图标：24vp × 24vp
- 大图标：32vp × 32vp
- 图标与文字间距：8vp
