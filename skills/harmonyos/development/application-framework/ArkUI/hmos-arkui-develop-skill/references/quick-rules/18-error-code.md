## 18. 错误码速查

| 错误码 | 说明 |
|--------|------|
| 100003 | Router 页面栈超过 32 页 |
| 100005 | Navigation 跳转未注册 builder 函数的页面 |
| 100006 | Navigation 跳转未包含 NavDestination 的页面 |
| 100021 | 声明式 FrameNode 不允许修改（设属性、增删子节点、绑定控制器） |
| 106203 | 对未挂载节点执行操作 |
| 130000 | addMonitor/clearMonitor 目标必须是 @ObservedV2 class 或 @ComponentV2 实例 |
| 130002 | addMonitor/clearMonitor 回调必须为命名函数 |
