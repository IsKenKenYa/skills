# 报错ERROR: unknown option '--serve'

**问题现象**

1. 点击启动按钮![start-logo](figures/start-logo.png)后长时间无反应，元服务无法正常启动。

2. 命令行启动热重载，报错信息为：error: unknown option '--serve'。

**可能原因**

ASCF ToolKit版本低于最低依赖版本要求。

**解决措施**


请将ASCF ToolKit版本升级至1.0.4以上，方法请参考：[升级已有项目的ASCF Toolkit版本](../quick-start/ascf-plugin.md#升级已有项目的ascf-toolkit版本)。
