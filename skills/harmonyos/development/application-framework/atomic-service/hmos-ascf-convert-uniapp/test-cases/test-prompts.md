**提示词**：
```
开发鸿蒙元服务需要安装哪些工具？版本有什么要求？
```

**预期输出**：
- 列出所需工具：DevEco Studio、HBuilderX（4.34+）

- 说明 ICP 备案的必要性

## 常见问题测试

### 测试场景 7：真机运行白屏问题排查
**提示词**：
```
我的鸿蒙元服务在真机上运行后是白屏，什么都没有，怎么排查？
```

**预期输出**：
- 给出 `hdc shell hilog` 日志查看命令及过滤参数
- 列出白屏常见原因：JS 运行时错误、首页路由配置错误、编译产物缺失、条件编译导致组件为空
- 说明如何通过日志定位具体错误
- 提醒检查 `pages.json` 中首页路径配置
- 说明 `hdc` 命令与 Android `adb` 命令的区别

### 测试场景 8：网络请求失败和域名白名单配置
**提示词**：
```
我的元服务里的网络请求都报错，提示网络异常，怎么解决？
```

**预期输出**：
- 说明鸿蒙元服务需要配置域名白名单
- 指导在 AGC 控制台添加请求域名的具体路径
- 说明开发和生产环境域名需要分别配置
- 提醒域名需使用 HTTPS，说明 HTTP 的限制情况
- 提供域名配置格式要求（不含路径、不含端口）

### 测试场景 9：定位权限申请问题
**提示词**：
```
我的元服务需要使用定位功能，但申请权限总是被拒绝，怎么处理？
```

**预期输出**：
- 说明需要在 `harmony-mp-configs` 的 `permissions` 中声明定位权限
- 给出定位权限的正确声明方式（精确定位和模糊定位权限名称）
- 说明运行时动态申请权限的代码方式（`uni.authorize` 或鸿蒙原生权限 API）
- 提醒 AGC 元服务配置中也需要声明所用权限
- 说明权限被用户拒绝后的降级处理建议

### 测试场景 10：获取手机号功能配置
**提示词**：
```
我想在鸿蒙元服务中使用华为账号获取手机号，需要怎么配置？
```

**预期输出**：
- 说明获取手机号需要开通华为账号登录能力
- 指导在 AGC 控制台开通相关服务
- 说明 `client_id` 配置要求
- 提供条件编译下调用华为账号获取手机号的代码示例
- 说明该功能仅在鸿蒙平台可用，需使用 `#ifdef MP-HARMONY` 包裹

## 编译报错测试

### 测试场景 11：签名配置错误 Invalid storeFile
**提示词**：
```
DevEco Studio 编译时报错 "Invalid storeFile"，我该怎么解决？
```

**预期输出**：
- 说明错误原因：签名文件路径配置错误（相对路径、路径不存在、Windows 反斜杠）
- 指导检查 `harmony-mp-configs.signingConfigs.storeFile` 的值
- 说明必须使用绝对路径（如 `/Users/xxx/xxx.p12`）
- 说明在 Windows 下路径分隔符应使用正斜杠 `/` 或双反斜杠 `\\`
- 提供正确的 `signingConfigs` 配置示例

### 测试场景 12：Unable to find product 'release' 报错
**提示词**：
```
编译报错 "Unable to find product 'release'"，请问怎么处理？
```

**预期输出**：
- 说明错误原因：`build-profile.json5` 中未配置 `release` 构建变体
- 说明 HBuilderX 发布操作会触发 release 构建，需提前配置
- 提供 `build-profile.json5` 中 release 构建变体的配置示例
- 说明 release 构建需要配置正式签名文件，不能使用自动调试签名
- 提示检查签名证书是否已过期

### 测试场景 13：install failed 安装失败报错
**提示词**：
```
元服务安装到设备时报错 "install failed"，是什么原因？
```

**预期输出**：
- 列出 install failed 的常见原因：包名不匹配、签名不匹配、设备存在旧版本冲突、HAP 包格式错误
- 指导先卸载设备上的旧版本再安装
- 说明检查包名是否与 AGC 元服务包名一致
- 说明签名证书的设备信任问题排查
- 提醒区分调试包和发布包的签名差异

### 测试场景 14：SDK 组件缺失和依赖模块报错
**提示词**：
```
编译时提示 "SDK component missing" 和 "dependent module not found"，怎么解决？
```

**预期输出**：
- 说明 SDK component missing 的解决方法：打开 DevEco Studio SDK Manager，找到缺失组件并安装
- 说明 dependent module not found 的常见原因：`oh-package.json5` 中依赖声明与实际安装不符
- 提供 `ohpm install` 命令重新安装依赖的操作
- 说明检查 DevEco Studio 版本与 SDK 版本兼容性
- 提醒在安装 SDK 组件后需要重新 sync 项目

## 上架审核测试

### 测试场景 15：图标不规范被驳回
**提示词**：
```
我提交元服务审核被驳回，说图标不规范，鸿蒙元服务图标有什么具体要求？
```

**预期输出**：
- 给出鸿蒙元服务图标尺寸要求（216x216 px）
- 说明图标必须使用圆角遮罩（系统自动应用，设计时应预留安全区）
- 列出图标设计禁止项：不含文字、不含边框、不与华为系统图标高度相似
- 说明图标格式要求（PNG，不含透明底）
- 建议参考 AGC 控制台的图标上传规范文档

### 测试场景 16：隐私政策缺失被驳回
**提示词**：
```
审核驳回说我没有隐私政策，隐私政策怎么配置和编写？
```

**预期输出**：
- 说明隐私政策需要配置可访问的 HTTPS URL
- 说明在元服务配置（`manifest.json` 或 AGC 控制台）中添加隐私政策链接的位置
- 列出隐私政策必须包含的核心内容（数据收集范围、存储方式、使用目的、第三方共享、用户权利）
- 提醒隐私政策页面必须使用 HTTPS 且全程可访问
- 说明华为审核对隐私政策的特殊要求（需说明华为账号相关数据处理）

### 测试场景 17：自行构造登录页被驳回
**提示词**：
```
审核驳回原因是"自行构造登录页"，这是什么意思？怎么修改？
```

**预期输出**：
- 解释"自行构造登录页"的含义：使用自定义 UI 模拟华为账号登录，而非调用官方华为账号登录 SDK
- 说明正确做法是使用华为账号授权登录 API（`uni.login` 配合鸿蒙平台或直接调用 HarmonyOS SDK）
- 提醒禁止在页面上展示用户名/密码输入框来模拟账号登录
- 说明使用条件编译 `#ifdef MP-HARMONY` 调用华为官方登录能力
- 提供正确的华为账号一键登录的调用方式参考

### 测试场景 18：设备类型不匹配被驳回
**提示词**：
```
审核驳回说设备类型不匹配，元服务支持的设备类型在哪里配置？
```

**预期输出**：
- 说明设备类型在 `build-profile.json5` 或 `app.json5` 中的 `deviceTypes` 字段配置
- 列出鸿蒙元服务支持的设备类型（phone、tablet 等）
- 说明 AGC 控制台提交时选择的设备类型需与配置文件一致
- 提醒元服务界面需适配所声明的设备类型（如声明支持 tablet 需做平板 UI 适配）
- 提供 `deviceTypes` 字段的配置示例
